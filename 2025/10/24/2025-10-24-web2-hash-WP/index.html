前四个IF

  • 条件 1：if( $val1 == $val2 ) → 触发 die

    • 在 PHP 中，两个不同内容的数组使用 == 比较时结果是 false，即使内容不同。
    • 所以：[1] == [2] → false

  • 条件 2：if( md5($val1) != md5($val2) ) → 要通过必须 md5($val1) == md5($val2)

    • PHP 无法对数组进行 md5() 计算，会发出警告并返回 NULL。
    • 所以:NULL != NULL → false

  • 条件 3：if( $val3 == $val4 )

    • 同 条件 1

  • 条件 4：if ( md5($val3) !== md5($val4))

    • !== 是 强比较（类型 + 值）
    • 因为两个 NULL 类型相同，值也相同，所以 !== 为 false
